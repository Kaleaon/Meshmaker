# Meshmaker

Meshmaker is an Android (Kotlin + Jetpack Compose) avatar creator for Second Life. It provides detailed anatomy sliders, swappable parts, and export-ready avatar/clothing packaging.

## Modules
- `app`: Android application module (Compose UI for avatar creation).
